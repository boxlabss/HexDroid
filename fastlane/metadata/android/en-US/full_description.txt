HexDroid is a clean, fast, open-source IRC client for modern Android. Connect to your favourite networks, chat with friends and communities, and never miss a mention—all from a Material Design interface built for mobile.

<b>Connect anywhere</b>
- Always Connected for persistent sessions and automatic reconnection
- Multiple network profiles with independent settings
- Auto-connect, auto-join and optional post-connect commands
- ZNC and soju bouncer support

<b>Security</b>
- TLS by default with Trust On First Use (TOFU) pinning for self-signed certificates
- SASL authentication: PLAIN, EXTERNAL (client certificates) and SCRAM-SHA-256
- Credentials encrypted with the Android Keystore
- Secure Chat with AES-256-GCM end-to-end encryption, Authenticated Group Exchange, FiSH for backwards compatibility.
- Tor and SOCKS proxy support

<b>Chat experience</b>
- Mention notifications with quick reply, swipe-to-reply and inline quote previews
- IRCv3 features including typing indicators, chat history, reactions, read markers and account tags
- /command and @nick autocomplete, colour-coded nicknames, mIRC/ANSI formatting and inline media previews
- Search, jump to unread, operator tools, ignore lists and notification muting

<b>DCC file transfers</b>
- Send and receive files, DCC Chat, passive/reverse DCC, Secure DCC (SSEND/SCHAT), RESUME and IPv6 support

<b>Customisation</b>
- Multiple languages and .hex scripting
- Per-network character encoding
- Custom nick colours and drag-and-drop network ordering
- Backup and restore settings as JSON
- Optional on-device chat logging

<b>Privacy</b>
No ads, no analytics and no tracking. Your messages and credentials are sent only to the IRC servers you choose.

IRCv3 features depend on server support. Background connectivity may vary depending on your device's battery settings.
